# 🚀 React Organo 🚀

📄 Descrição do Projeto

Este é um projeto desenvolvido com ReactJS, com o objetivo de praticar a criação de componentes reutilizáveis e a manipulação de estados e props. O Organo simula uma aplicação de gestão de equipes, permitindo adicionar colaboradores e organizá-los por times.

No desenvolvimento deste projeto foram aplicados conceitos importantes como:

✅ Criação de componentes funcionais com React.

✅ Uso de props para comunicação entre componentes.

✅ Manipulação de estados com useState.

✅ Renderização dinâmica de elementos com .map().

✅ Estilização com CSS Modules para isolamento de estilos.


🚀 Tecnologias utilizadas

ReactJS

JavaScript (ES6+)

HTML5

CSS3 (CSS Modules)

🎯 Objetivos alcançados

Entender a estrutura e o fluxo de dados no React.

Praticar a criação de componentes reutilizáveis.

Aplicar a renderização condicional e listas dinâmicas.

Publicar o projeto utilizando o Vercel.
